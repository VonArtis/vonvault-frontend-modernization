### 📦 **Upload Instructions**

1. **Create/Clone Repository**
   ```bash
   git clone https://github.com/VonArtis/vonvault-frontend-modernization.git
   cd vonvault-frontend-modernization
   ```

2. **Copy Files**
   Copy all files from `/app/frontend/` to your local repository

3. **Initial Setup**
   ```bash
   # Install dependencies
   yarn install
   
   # Set up environment
   cp .env.example .env
   # Edit .env with your backend URL
   
   # Start development
   yarn start
   ```

4. **Push to Repository**
   ```bash
   git add .
   git commit -m "Initial commit: VonVault frontend modernization"
   git push origin main
   ```
